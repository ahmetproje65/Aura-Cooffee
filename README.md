# ☕ Aura Roastery | Coffee Empire Platform

![Aura Roastery Banner](https://images.unsplash.com/photo-1497935586351-b67a49e012bf?q=80&w=1920&auto=format&fit=crop)

Aura Roastery is a premium, high-fidelity web platform designed for a global coffee brand. It combines a stunning corporate presence with a fully functional online shop and a real-time administrative management panel.

## 🚀 Features

### 💎 Premium Frontend (index.html)
- **Glassmorphism Design:** Modern, sleek UI with smooth scrolling and interactive hover effects.
- **Hero Video Background:** Cinematic coffee preparation visuals.
- **Franchise System:** Interactive partnership and investment sections.
- **Live Support Widget:** Real-time bridge between customers and the admin panel.

### 🛒 Online Store (shop.html)
- **Dynamic Product Grid:** 30+ products across Coffees, Chocolates, and Gift categories.
- **Advanced Cart System:** Real-time price calculation and offcanvas cart management.
- **Secure Checkout:** Simulated checkout process with unique order ID generation.
- **Live Order Tracking:** Interactive modal showing progress from "Received" to "Delivered".

### 🛠️ Admin Panel (admin.html)
- **Live Dashboard:** Real-time sales charts (Chart.js) and active order monitoring.
- **Order Management:** Ability to approve, cancel, or view details of incoming web orders.
- **Real-time Support Center:** Admin can reply to customer support requests instantly.
- **Operational Settings:** Control shop status, courier assignments, and sound alerts.

## 🛠️ Technology Stack
- **Languages:** HTML5, Vanilla JavaScript (ES6+), CSS3
- **Frameworks:** Bootstrap 5 (Styling), Chart.js (Data Visualization)
- **Icons & Fonts:** FontAwesome 6, Google Fonts (Playfair Display & Inter)
- **Animations:** CSS Keyframes & Intersection Observer API
